This project demonstrates the integration of Flask and the Wikipedia API to create a dynamic web application. Users can search for Wikipedia pages and view their titles, summaries, and links, all formatted neatly using Jinja templates.

Features

  • Search Wikipedia directly from the web app
  • Formatted results with page titles, summaries, and URLs
  • Template inheritance for consistent navigation and footer

Resources Used

  • Flask (Python web framework)
  • Jinja2 for templating
  • Wikipedia API for fetching data
